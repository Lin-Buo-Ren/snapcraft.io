Why are snaps good for Python projects?

  • The Snap Store makes your app easy to discover and install for millions of users, graphically or at the command line
  • Automatic updates migrate your users to the latest stable version
  • If an update fails, the previous version is reverted, preserving data
  • Isolation prevents conflicts between applications
  • Identical behaviour across Linux distributions, even with C library dependencies

With PyPi you can distribute Python apps, and virtualenv gives you a solution to app and system dependencies. Snaps combine these ideas, offering a Linux app store experience for end users.

Here’s how offlineimap uses it:

name: offlineimap
version: git
summary: OfflineIMAP
description: >
  OfflineIMAP is software that downloads your email mailbox(es) as local
  Maildirs. OfflineIMAP will synchronize both sides via IMAP.

confinement: devmode

parts:
  offlineimap:
    plugin: python
    python-version: python2
    source: .
    stage-packages:
      - python-six

apps:
  offlineimap:
    command: bin/offlineimap
